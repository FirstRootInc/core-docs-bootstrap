The total reactions received by a `Reactionable` object.

This type contains the total number for every single reaction type.

---

**Example:**

| Payload                                | Description                                                                                                                         |
| :------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------- |
| `{likes: 10, dislikes: 1, neutral: 3}` | This proposal has received **10** reactions of type `LIKE`, **1** reaction of type `DISLIKE` and **3** reactions of type `NEUTRAL`. |

```graphql
type Reactions {
  dislikes: Int!
  likes: Int!
  neutral: Int!
}
```
