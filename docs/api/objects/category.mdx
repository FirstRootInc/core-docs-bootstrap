### Fields

#### `color` ([`String`](/docs/api/scalars/string))

The color assigned to the category. This color can be used by clients to customize the presentation of the category, for instance, to change the background color, title color, etc.

The returned value is formatted as web hex color (e.g. `#FFFFFF`).

**Access level:** `public`.  
**Returns:** `String` | `null`.

#### `hexColor` ([`String`](/docs/api/scalars/string))

The color assigned to the category. This color can be used by clients to customize the presentation of the category, for instance, to change the background color, title color, etc.

The returned value is formatted as an hex string (e.g. `0x75dddd`).

**Access level:** `public`.  
**Returns:** `String` | `null`.

#### `icon` ([`String`](/docs/api/scalars/string))

The emoji icon assigned to the category.

**Access level:** `public`.  
**Returns:** `String` | `null`.

#### `id` ([`ID!`](/docs/api/scalars/id))

The category `UUID` identifier in the system.

**Access level:** `public`.

#### `proposals` ([`[Proposal!]`](/docs/api/objects/proposal))

All proposals assigned to this category. More information about this field can be found in the description of `ProposalType`. Access to this field requires a logged in user, anonymous requests will always return `null`.

**Access level:** `public`.  
**Returns:** `ProposalType` | `null`.

#### `rgbColor` ([`[Int!]`](/docs/api/scalars/int))

The color assigned to the category. This color can be used by clients to customize the presentation of the category, for instance, to change the background color, title color, etc.

The returned value is formatted as an array of integers where every index represents a color code for [`red`, `green`, `blue`] respectively.

**Access level:** `public`.  
**Returns:** `[red, green, blue]` | `null`.

The category title provided by participant.
