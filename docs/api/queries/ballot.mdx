Query to retrieve a ballot infomration. Notice that you must provide at least one argument `pbCycleId` or `ballotId`. Consider `ballotId` has precedence over `partyBallotId`.

```graphql
ballot(
  pbCycleId: String
  ballotId: String
): Ballot!

```

### Arguments

#### `pbCycleId` ([`String`](/api/scalars/string))

The PB cycle ID which the ballot belongs to. If a `ballotId` is provided then this value will be ignored.

#### `ballotId` ([`String`](/api/scalars/string))

The specific ballot ID to retrieve its information. When it is provided the `pbCycleId` field is ignored.

### Type

#### [`Ballot`](/api/objects/ballot)
