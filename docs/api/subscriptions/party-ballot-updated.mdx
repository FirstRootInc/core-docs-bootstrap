Subscription to retrieve a party ballot updates. This subscription requires the user is logged in and user represents a party owner of the party ballot specified in the **`partyBallotId`** argument.

```graphql
partyBallotUpdated(
  partyBallotId: ID!
): PartyBallotUpdatedPayload!

```

### Arguments

#### `partyBallotId` ([`ID!`](/docs/api/scalars/id))

The party ballot ID. This ballot must belong to current user.

#### [`PartyBallotUpdatedPayload`](/docs/api/objects/party-ballot-updated-payload)

Autogenerated return type of PartyBallotUpdated
