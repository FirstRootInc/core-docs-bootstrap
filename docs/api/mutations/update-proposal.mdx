Allows to update a proposal under next restrictions:

- Only participant who creates this proposal can edit it.
- The proposal can be assigned only to a PB Cycle in which the participant is allowd to participate.
- Only provided arguments will be updated.

  input: UpdateProposalInput!
): UpdateProposalPayload
